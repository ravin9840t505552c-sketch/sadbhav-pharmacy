# Sadbhav Pharmacy — Website

Sadbhav Pharmacy is a small, patient‑centered pharmacy located in **Pipalchowk, Bharatpur‑4, Chitwan, Nepal**.  
This website is a **static, frontend‑only** site with no backend, built in pure HTML, CSS, and JavaScript.

---

## 📌 Project Overview

- **Purpose:**  
  Provide an online presence for Sadbhav Pharmacy, allowing visitors to learn about services, view the team, and send prescriptions via Cloudinary + WhatsApp.

- **Key Features:**
  - Responsive layout on desktop and mobile.
  - No backend code or server‑side logic.
  - SEO‑optimized meta tags and Open Graph / Schema‑markup.
  - Prescription upload to **Cloudinary** (no email server needed).
  - User can share the Cloudinary URL via **WhatsApp** or copy‑paste into email.

- **Stack:**
  - HTML
  - CSS (`css/style.css`)
  - JavaScript (`js/script.js`)
  - CSS‑only animation (no extra frameworks)
  - [Lucide Icons](https://lucide.dev) via CDN
  - [Cloudinary](https://cloudinary.com) image‑upload (unsigned)
